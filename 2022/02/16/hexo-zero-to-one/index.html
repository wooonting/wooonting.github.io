hexo zero to one

安装、初始化环境

install Node.js //安装node.js
npm install -g hexo //安装hexo
hexo init //在一个文件夹下，做hexo初始化

测试使用HEXO

hexo generate //生成静态页面
hexo server //本地启动hexo服务器

关联Hexo与Github

deploy:
type: ‘git’
repository: https://github.com/wooonting/woooting.github.io.git
branch: master

安装插件hexo-deployer-git

npm install hexo-deployer-git –save //使用包管理工具npm安装插件，用于发布Hexo到github上（主要做的工作是：把Public文件夹下的文件复制到.deploy_git下），问题是public下的文件怎么来的？

将hexo目录部署到github上

hexo clean
hexo generate // 生成静态页面
hexo deploy // 部署到github

新建文章

hexo new post “hexo-zero-to-one”
hexo generate // 生成静态页面
hexo deploy // 部署到github
