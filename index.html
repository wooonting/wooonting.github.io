DNS

DNSDomain Name Server 域名服务器 DNS 的作用DNS负责记录IP地址与域名的对应关系。将域名解析成IP地址，方便主机之间通过IP地址来传输数据包。 正向解析输入域名，得到IP（网络传输底层需要解析成IP地址） 反向解析输入IP，得到域名。没有注册过合法域名的、没有向ISP申请IP反向解析的邮件发送者，一般会被邮件服务器/邮件网关的“反垃圾邮件功能”判断为垃圾邮件。
